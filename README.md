# FitToMarket - AI Resume Analyzer

FitToMarket is an AI-powered tool designed to analyze resumes and provide actionable feedback to help job seekers improve their chances of getting hired.

## Features

- **Resume Analysis**: Upload your resume (PDF/DOCX) for detailed analysis.
- **Job Role Matching**: Get feedback tailored to specific job roles.
- **AI Scoring**: Receive a score based on industry standards.
- **Recommendations**: Actionable tips to improve formatting, content, and keywords.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/FitToMarket.git
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Set up environment variables in `.env` (API keys, etc.).
4.  Run the application:
    ```bash
    python run.py
    ```

## Usage

Navigate to `http://localhost:5000` in your browser. Upload your resume and view the results.
